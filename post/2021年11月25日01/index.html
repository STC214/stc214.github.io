docker镜像：dezhao/ubooquity

拉取镜像

docekr pull dezhao/ubooquity

使用镜像

1.文件夹配置

需要配置config 和漫画文件夹
根据个人需求可以配置电子书目录


-v /<docker外路径> : /<docker内路径>

如：

-v /home/ubooquity/config:/config
-v /home/ubooquity/comics:/comics

更换主题只需要在config/themes文件夹中解压即可识别。

2.端口配置

2202 2203


-p /<docker外端口> : /<docker内端口>

如：

-p 2202:2202

其他配置可以根据个人需求微调

3.运行命令：

如有图形界面仅需要根据个人情况配置即可完成


docker run -d -t \
--restart unless-stopped \
--name ubooquity \
-p 2202:2202 \
-p 2203:2203 \
-v <path to data>:/config \
-v <path to books>:/books \
-v <path to comics>:/comics \
-e TZ=Asia/Shanghai \
-e MAXMEM=1.5G \
dezhao/ubooquity

使用配置

1. 后台

https://<host_ip>:2203/admin

1> 进入后首先配置漫画选项卡
2> 将配置为漫画存放目录的路径点击分享
3> 勾选页面最上方的分享选项
4> 保存并重启

2. 扫描

等待一般选项卡中扫描完全部内容后方可改变主题（未扫描完时改变主题后点击应用并重启会导致扫重头再来）

3. 正常使用

https://<host_ip>:2202
